NOn-invasive neonate jaundice estimation using artificial intelligence

NINJA

Non-invasive methods for jaundice detection include visual assessments, transcutaneous bilirubinometers, and optical biosensors. These methods aim to alleviate the need for invasive blood tests, which can be painful for newborns. They also help expedite the turnaround time for obtaining results and reduce the risk of iatrogenic anemia.

DATASET ANNOTATIONS

Annotation Tools

An open-source tool, DarkLabel is used to annotate the videos. Click on the link provided to know more about it.

The Process

All the videos are converted into frames and the annotations are provided for each frame. The annotations are mapped back to the original images for validating the dataset.

Representation

Annotations are represented with the help of seven attributes:

  • Name of object's class
  • Action of Object
  • Object's ID
  • Bounding box Coordinates (x,y,w,h)
